# Recipes + Pantry

A gathering of recipes, collected and modified with love.

### Data files

`pantry.json`: a JSON representation of ingredients/pantry items, with nutrition and price information. This is user inputted.<br />
`generated-pantry.json`: this is a JSON generated by the `calculate.py` script, with extra useful pantry info<br />
`recipes.json`: a JSON representation of recipes, their ingredients and directions. This is user inputted.<br /><br />
`recipes-with-nutrition.json`: this is a JSON generated by the `calculate.py` script, with all recipes' nutrition and price info<br />

### Data processing

The `calculate.py` script goes through all items in `pantry.json`, calculating all their nutrition and price information per 100 grams.<br />
It also goes through all recipes in `recipes.json`, calculating all their nutrition and price information based on the information from `generated-pantry.json`.<br />
`python calculate.py` to run this script
